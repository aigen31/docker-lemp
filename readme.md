# Lightweight and Universal LEMP Build on Docker for Fast Site Development and Deployment

## Get Started:
```sh
sudo ./run.sh
```

## Supported Image Versions

| Service  | Image          | Version  |
|----------|----------------|----------|
| Nginx    | nginx          | latest   |
| PHP      | php            | 8.3-fpm  |
| MySQL    | mysql          | 8.0      |
